# @amlplugins/replicate-trainings

Native `replicate` SDK re-exported under the `@amlplugins` namespace.

> Replicate Trainings — fine-tune SDXL, FLUX, and Llama with custom datasets via training jobs.

## Install

```bash
npm install @amlplugins/replicate-trainings
```

## Use

```ts
import { /* named exports from replicate */ } from "@amlplugins/replicate-trainings";
// or
import sdk from "@amlplugins/replicate-trainings";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
